## UntitledDesktop
### UntitledCLIParser 6.0 and UntitledImGuiFramework 2.5
The 2.5 release of the [UntitledImGuiFramework](https://github.com/MadLadSquad/UntitledImGuiFramework) is mainly upstreaming version 6.0 of the [UntitledCLIParser](https://github.com/MadLadSquad/UntitledCLIParser).

The new 6.0 release of the UntiltedCLIParser library is a complete rewrite of the library. The main difference is in the API, where the old event-based API is replaced by a tree-based API. This tree-based API allows
for better automatic handling of arguments, better argument validation, automatic generation of help messages and requires less manual handling inside each event.

Both releases should land aroudn the 10th of February.
